Exercício 2: Formatação de Texto e Blocos Pré-formatados

Formatação de Texto

Texto em negrito com <b> - usado apenas para estilo visual, sem importância semântica.

Texto importante com <strong> - além do estilo visual, indica que o conteúdo tem forte importância ou urgência.

Texto em itálico com <i> - usado para voz alternativa, termos técnicos, pensamentos, etc.

Texto com ênfase usando <em> - indica que as palavras devem ter ênfase na pronúncia quando lidas.

Texto com palavra destacada usando a tag <mark> - como um marca-texto.

Texto menor usando <small> - ideal para notas de rodapé, avisos legais, etc.

Texto com palavra removida e palavra inserida - usando <del> e <ins> para mostrar edições.

Fórmula química: H2O - usando <sub> para subscrito.

Equação matemática: X2 + Y2 = Z2 - usando <sup> para sobrescrito.

Bloco de Texto Pré-formatado

A tag <pre> mantém os espaços em branco e as quebras de linha exatamente como foram escritas no código-fonte, preservando a formatação original.
